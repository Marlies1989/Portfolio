About

Projects

Website redesign for Logement Dwarsgracht, because the old one was not responsive.

Logement Dwarsgracht started in 2008, this was also the time the current website was built. At that time responsive webdesign was not there yet. These days your website can not without responsiveness.

The new website is build with Vue.js, because I wanted to learn to build a simple website with Vue.js. And offcourse it is responsive.

Currently, this website is not live yet but is deployed with Netlify. Netlify is used so the customer can see and test a live preview of the new website and give their feedback.
